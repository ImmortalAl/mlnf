How to Test

  1. Open the browser's developer tools (F12 or right-click → Inspect)
  2. Go to the Console tab
  3. Click on the test links above
  4. Check for any errors in the console

Running a Local Server

To test these files locally, you can use Python's built-in HTTP server:

# Python 3
python -m http.server 8000

Or use the provided PowerShell script:

# In PowerShell
.\start-server.ps1

Then open http://localhost:8000/test-local.html in your browser.
